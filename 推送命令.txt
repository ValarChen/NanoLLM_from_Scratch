GitHub推送命令（按顺序执行）

========================================
1. 如果还没有设置远程仓库，先添加：
========================================
git remote add origin https://github.com/你的用户名/NanoLLM_from_Scratch.git

（将"你的用户名"替换为你的GitHub用户名）

========================================
2. 然后推送代码：
========================================
git push -u origin main

========================================
3. 如果已经设置了远程仓库，直接推送：
========================================
git push

========================================
注意事项：
========================================
- 如果提示需要认证，使用GitHub个人访问令牌（Personal Access Token）作为密码
- 生成Token：GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
- 如果推送失败，检查网络连接和仓库权限
