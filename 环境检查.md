## ⚠️ 注意事项

### PATH 警告

安装过程中出现了一些警告，提示 Scripts 目录不在 PATH 中：
```
C:\Users\Administrator\AppData\Local\Python\pythoncore-3.14-64\Scripts
```

**影响**：
- 不影响使用 `python -m` 运行脚本
- 不影响项目运行
- 只是不能直接使用 `pip`、`tensorboard` 等命令

**解决方案（可选）**：
如果需要直接使用这些命令，可以：
1. 将 Scripts 目录添加到系统 PATH
2. 或继续使用 `python -m pip`、`python -m tensorboard` 等方式

## 📝 使用说明

由于 Scripts 目录不在 PATH 中，建议使用以下方式：

- 安装包：`python -m pip install <package>`
- 运行脚本：`python -m src.train`
- 运行 TensorBoard：`python -m tensorboard.main --logdir=logs`

所有项目功能都可以正常使用！
